A RESTful backend for an e-commerce application built using Spring Boot, Spring Security, and PostgreSQL.

# Spring Boot E-Commerce Backend

## 📌 Project Description

This is a **Spring Boot based E-Commerce backend application** that provides REST APIs for managing products, users, authentication, and orders.
The application uses **PostgreSQL** as the database and can be deployed on **AWS Elastic Beanstalk**.

---

## 🚀 Features

* User Authentication (JWT)
* Product Management
* Category Management
* Order Management
* Role-based authorization
* RESTful APIs
* PostgreSQL database integration

---

## 🛠 Tech Stack

* Java 17
* Spring Boot
* Spring Security
* Spring Data JPA
* PostgreSQL
* Maven
* AWS Elastic Beanstalk
* Amazon RDS

---


## 🌐 Deployment

The application is deployed using:

* AWS Elastic Beanstalk
* Amazon RDS (PostgreSQL)
